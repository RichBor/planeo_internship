HTML & CSS Grundstrukturen

Eine standart HTML-Struktur besteht aus einem Head, einem Body und einem Style, der entweder im Head zu finden oder in einer seperaten Datei verlinkt ist.

Der Head



Der Head enthaelt Metadaten wie Keywords, Autor, verlinkte Stylesheets oder Seitentitel.
Diese Daten werden z.B von Browsern und Suchmaschinen benutzt.

Head von W3Schools.com
Der Body



Der Body enthält das ganze Grundgerüst. Hier steht der ganze Content.

-------------------------------------------------------------

Der Style



Der Style enthält idealerweise das ganze Design.
Es ist entweder im Head direkt, oder verlinkt zu sehen.
Das Ziel, den Style von HTML direkt zu trennen ist, es übersichtlicher zu halten.

----------------------------------------------------------------------

Auf dieser Seite ist der Style in einer seperaten Datei angelegt. Diese Datei kann man im Head verlinken und so in das HTML einbinden

HTML Text Structure

Paragraphen



Ein Paragraph ist ein block-Element um Text zu beschreiben.

ungeordnete Listen



Eine ungeordnete Liste ist einfach eine Liste mit Stichpunkten, also keiner direkten Reihenfolge:

  • Mercedes
  • Audi
  • VW
  • Skoda
geordnete Listen



Eine geordnete Liste ist eine Liste, die durchnummeriert ist.
Das ist praktisch für z.B. Schritt für Schritt Anleitungen.

  1. Eier einschlagen
  2. Mehl und Butter hinzufügen
  3. Umrühren
  4. verteilen
div's



Ein div ist ein block-Element, mit dem man einen Container erschaffen kann (wie eine box).
Das macht man um bestimmte Sektionen in Websiten einzuteilen und sie dann individuell ansprechen zu können.
